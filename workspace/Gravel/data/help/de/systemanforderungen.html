Systemanforderungen

„Gravel“ benötigt mindestens Java 1.5.0 (Sun JDK 5). Es mag mit den vorherigen Versionen starten, jedoch wird der volle Funktionsumfang eher nicht verfügbar sein. Mit OpenJDK wurde „Gravel“ nocht nicht getestet.

Zum Laden von Graphen ist eine Internet-Verbindung notwendig, da die XMLSchema_Datei zur Validierung bisher nur online verfügbar ist. Das hat den Grund darin, dass so kleinere Fehler ebenso wie Erweiterungen nur an dieser Datei vorgenommen werden müssen und dann allen zur Verfügung stehen. Ein weiterer Grund ist, dass die Basis, also das graphml-Schema ebenso online verfügbar ist.
